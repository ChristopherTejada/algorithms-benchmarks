# Algorithms Benchmarks

This repository contains implementations of classic algorithms and
data structures, focusing on performance analysis and benchmarking.

## Goals
- Implement fundamental algorithms in a clean and readable way
- Measure and compare execution time
- Analyze algorithmic complexity in practice

## Topics
- Sorting algorithms
- Graph algorithms
- Performance benchmarking
- Complexity comparison

## Structure
- `sorting/` – Sorting algorithm implementations
- `graphs/` – Graph algorithms
- `benches/` – Benchmark scripts
- `plots/` – Generated performance charts

## How to run
Benchmarks are executed using Python scripts located in `benches/`.

# Benchmarks de Algoritmos

Este repositorio contiene implementaciones de algoritmos clásicos y
estructuras de datos, con un enfoque en el análisis de rendimiento y benchmarking.

## Objetivos
- Implementar algoritmos fundamentales de manera clara y legible
- Medir y comparar tiempos de ejecución
- Analizar la complejidad algorítmica en la práctica

## Temas
- Algoritmos de ordenamiento
- Algoritmos de grafos
- Benchmarking de rendimiento
- Comparación de complejidad

## Estructura
- `sorting/` – Implementaciones de algoritmos de ordenamiento
- `graphs/` – Algoritmos de grafos
- `benches/` – Scripts de benchmarking
- `plots/` – Gráficos de rendimiento generados

## Cómo ejecutar
Los benchmarks se ejecutan utilizando scripts en Python ubicados en `benches/`.
